# 🌍 Dağınık Gezgin

A colorful and educational children's game project designed to encourage exploration, curiosity, and interactive learning through playful environments and engaging visual design.

---

## 🎮 Project Overview

**Dağınık Gezgin** is a children's game concept developed as part of a professional project collaboration.

The project focuses on:

- Exploration-based gameplay
- Child-friendly visual storytelling
- Interactive environments
- Educational and entertaining game mechanics
- Colorful and accessible design language

This repository currently serves as a **showcase and presentation page** for the project.

---

## ✨ Features

- 🌈 Cartoon-style visual design
- 🧩 Interactive gameplay concepts
- 🗺 Exploration-focused world structure
- 👶 Child-oriented user experience
- 🎨 Custom UI/UX ideas
